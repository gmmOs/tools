####FLAGS
NS="ns" # - Nameserver records contain the name of the authoritative servers hosting the DNS records for a domain.
A="a" # - Also known as a host record, the "a record" contains the IP address of a hostname (such as www.megacorpone.com).
MX="mx" # - Mail Exchange records contain the names of the servers responsible for handling email for the domain. A domain can contain multiple MX records.
PTR="ptr" # - Pointer Records are used in reverse lookup zones and are used to find the records associated with an IP address.
CNAME="cname" # - Canonical Name Records are used to create aliases for other host records.
TXT="txt" # - Text records can contain any arbitrary data and can be used for various purposes, such as domain ownership verification.
